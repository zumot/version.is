version.is

| An API for getting the latest version of Open Source projects

Get Version Info

Usage

Get the version of a single project: 

http://version.is/<project_name>

Get a list of all projects monitored:

http://version.is/projects

The response can be delivered in 3 different formats; text/html, text/plain, application/json. You can specify the format using the querystring like this: 

http://version.is/?format=<html|plain|json>
http://version.is/projects?format=<html|plain|json>

Another way to specify the format, is using HTTP-Accept Headers. Consider the below example of fetching current jquery version in json format with curl in a terminal.
(Note that formats defined in the querystring overrules the HTTP-Accept header. For json requested with HTTP-accept header, you still need to use the querystring for a JSONP-callback, as described below.) 

curl -i -H "Accept: application/json" "http://version.is/jquery"

If you request the json format, you can specify a callback for JSONP, like this: 

http://version.is/<project_name>?format=json&callback=<callbackFunctionName>
